# Vital Signs Monitoring System (C++)
A system to monitor and log the vitals of simulated patient.

# Features

• Simulated patient vital signs

• Abnormal detection

• Real-time monitoring

• CSV logging

# Vitals monitored
• Heart Rate

• SpO₂

• Respiration Rate

• Blood Pressure

# Technologies
• C++

• multithreading

• file I/O

• real-time simulation
